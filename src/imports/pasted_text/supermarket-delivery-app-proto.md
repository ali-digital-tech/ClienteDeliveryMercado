Crie o protótipo de um aplicativo mobile de delivery de supermercado, focado na experiência do cliente final que vai fazer compras pelo celular. O app deve transmitir praticidade, confiança, organização e agilidade. A interface precisa parecer de um supermercado grande e profissional, com visual moderno, limpo e intuitivo.

Objetivo do app

O aplicativo serve para que o cliente:

encontre produtos do supermercado
navegue por categorias e ofertas
adicione itens ao carrinho
finalize a compra
escolha entrega ou retirada
acompanhe o pedido até a entrega
Estilo visual

Quero um design:

moderno
clean
mobile-first
com aparência profissional de grande supermercado
fácil de usar
com foco em conversão
visual organizado, sem excesso de informação
com cards bonitos para produtos
banners promocionais bem destacados
sensação de app confiável e rápido
Direção de UI

A interface deve ter:

header limpo
busca muito visível
categorias com ícones
cards de produto com imagem, nome, preço e botão de adicionar
destaque forte para promoções
carrinho simples e intuitivo
checkout claro e objetivo
tela de acompanhamento do pedido com timeline/status
Estrutura da aplicação do cliente
1. Splash Screen

Tela inicial com logo do supermercado, fundo elegante e loading simples.

2. Tela de Boas-vindas / Localização

Tela para o cliente informar ou confirmar localização.
Elementos:

mensagem de boas-vindas
botão para usar localização atual
campo para informar CEP ou endereço
aviso sobre entrega na região
3. Home do App

Essa é a principal tela do cliente.
Quero uma home rica, organizada e com foco em compra rápida.

Elementos da home:

topo com logo, localização e ícone do carrinho
barra de busca destacada
banner principal de promoções
seção de categorias com ícones
seção “Ofertas do dia”
seção “Mais vendidos”
seção “Compre novamente”
seção “Produtos em destaque”
cards de produto com:
imagem
nome
preço atual
preço promocional se houver
botão de adicionar
navegação inferior fixa

A home deve parecer viva e comercial, como um app que incentiva a compra.

4. Tela de Categorias

Tela mostrando todas as categorias do supermercado.
Exemplos:

Hortifruti
Açougue
Bebidas
Padaria
Laticínios
Limpeza
Higiene
Mercearia
Congelados
Pet

A tela deve ter grid ou lista visual bonita, com ícones ou imagens representativas.

5. Tela de Busca / Resultados

Tela para mostrar resultados de busca de produtos.
Elementos:

campo de busca no topo
filtros
ordenação
lista de produtos
opção de filtrar por promoção, menor preço, marca e categoria
6. Tela de Detalhe do Produto

Tela completa de um produto.

Elementos:

imagem grande do produto
nome
marca
descrição curta
preço normal
preço promocional
seletor de quantidade
botão de adicionar ao carrinho
informações complementares
seção de produtos relacionados

A tela deve passar confiança e facilitar a decisão de compra.

7. Tela de Favoritos

Tela com produtos salvos pelo cliente.
Layout simples, com cards e botão para adicionar ao carrinho novamente.

8. Tela de Carrinho

Tela muito importante, com design claro e objetivo.

Elementos:

lista dos produtos adicionados
imagem pequena do item
nome
quantidade com botões de aumentar e diminuir
preço por item
botão de remover
campo para cupom de desconto
resumo com subtotal
desconto
taxa de entrega estimada
total
botão principal “Continuar para entrega”

O carrinho deve parecer simples, rápido e sem confusão.

9. Tela de Login / Cadastro

Tela para o cliente entrar na conta ou criar cadastro.

Elementos:

login com telefone ou e-mail
campo de senha
opção de criar conta
opção de recuperação de senha
visual simples e confiável
10. Tela de Endereços

Tela para escolher ou cadastrar endereço.

Elementos:

lista de endereços salvos
botão para adicionar novo endereço
campos para rua, número, bairro, complemento, referência
opção de marcar endereço principal
11. Tela de Entrega ou Retirada

Tela para o cliente escolher como quer receber o pedido.

Opções:

entrega em casa
retirada na loja

Mostrar:

endereço selecionado
previsão de entrega
valor da taxa
horário estimado
unidade para retirada, se aplicável
12. Tela de Checkout

Tela final antes de pagar.

Elementos:

resumo do pedido
lista resumida dos itens
endereço
tipo de entrega
forma de pagamento
subtotal
taxa de entrega
desconto
total final
botão “Finalizar pedido”
13. Tela de Pagamento

Tela para selecionar a forma de pagamento.

Opções:

PIX
cartão
dinheiro
pagamento na retirada, se aplicável

Mostrar interface clara, organizada e simples.

14. Tela de Pedido Confirmado

Tela de sucesso após finalizar compra.

Elementos:

ícone de confirmação
número do pedido
mensagem de sucesso
prazo estimado
botão para acompanhar pedido
botão para voltar à home
15. Tela de Acompanhamento do Pedido

Tela mostrando o status do pedido em tempo real.

Status sugeridos:

pedido recebido
pedido confirmado
em separação
saiu para entrega
entregue

Quero essa tela com uma timeline visual bonita e fácil de entender.

16. Tela de Meus Pedidos

Tela com histórico de compras do cliente.
Elementos:

lista de pedidos anteriores
data
valor
status
botão para ver detalhes
botão para repetir pedido
17. Tela de Perfil do Cliente

Tela com informações da conta.

Elementos:

nome
telefone
e-mail
endereços
favoritos
métodos de pagamento
meus pedidos
suporte
sair da conta
Navegação do app

Criar navegação inferior com 4 ou 5 itens:

Home
Categorias
Favoritos
Pedidos
Perfil

O carrinho pode ficar em destaque no topo ou com acesso rápido.
